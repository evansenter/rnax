**RNAx** ~ An end-to-end JS platform for managing and deploying webtools with long running server-side tasks.
--------------------------------------------------------------------------------------------------

### Technologies used:
* Node.js
* Heroku (prod. only)
* Supervisor (dev. only)
* Express
* Backbone.js
* Mongodb
* Mongoose
* Jade
* Stylus
* Underscore.js
* _...and probably others, look at package.json over that way if you need the deets._
